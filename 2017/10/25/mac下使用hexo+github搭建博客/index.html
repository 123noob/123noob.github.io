mac下使用hexo+github搭建博客

使用homebrew安装node.js

github新建项目，项目命名遵守格式:账户名.github.io。建好后你会发现外网可以访问它。

在本地计算机安装hexo并初始化

键入npm install，安装所有所需组建
键入hexo g，首次体验hexo
键入hexo s，开启服务器，也可以使用“hexo server -p 端口”命令来改变端口

将本地hexo项目和github关联起来

如果在显示id_rsa id_rsa.pub，说明生成成功！

登录github，点击settings->SSH and GPG keys-> New SSH key,将id_rsa.pub文件里的内容复制上去保存

在终端 ssh -T git@github.com测试

配置Deployment，在其文件夹中，找到_config.yml文件，修改repo值

本地新建一篇博客，“hexo new post 博客名”，会在source/_posts下生成文件

在生成以及部署文章之前，需要安装一个扩展“npm install hexo-deployer-git –save”

接着命令“hexo g -d”,http://账户名.github.io。那么将看到生成的这篇文章。
